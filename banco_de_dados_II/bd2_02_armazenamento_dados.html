IFMG 35
iSCSI
Internet Small Computer System Interface (iSCSI)
Protocolo que permite transportar blocos de dados SCSI sobre redes IP
Possibilita que servidores acessem dispositivos de armazenamento remotos como se
estivessem conectados localmente
Comparado ao Fibre Channel, o iSCSI uma alternativa mais econmica e escalvel
O iSCSI frequentemente utilizado em ambientes de virtualizao e armazenamento
centralizado
